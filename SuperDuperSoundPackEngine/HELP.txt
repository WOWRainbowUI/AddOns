This addon will add sounds to the game from compatible sound packs.

You may use the "Super Mario World, of Warcraft" addon as an API template:
http://wow.curse.com/downloads/wow-addons/details/supermarioworldofwarcraft.aspx

To update a translation please use the localization utility at:
http://wow.curseforge.com/addons/superdupersoundpackengine/localization/


Slash commands to open the option window:
/soundpack
/superduper
/sdspe

Slash command to disable and enable all sounds:
/soundpack sound
